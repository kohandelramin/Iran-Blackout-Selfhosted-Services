# چک لیست قبل از اختلال

این کارها را قبل از زمانی انجام بده که واقعا به سرویس‌ها نیاز داری.

## سرور

- Docker نصب باشد
- Docker Compose کار کند
- Nginx نصب باشد
- UFW فعال باشد
- ورود با SSH key تست شده باشد
- اگر امکانش هست password login غیرفعال باشد
- بکاپ‌ها تست شده باشند

## ایمیج‌ها

تا وقتی اینترنت بهتر است ایمیج‌ها را pull کن:

```bash
docker pull filebrowser/filebrowser:latest
docker pull owncast/owncast:latest
docker pull teamspeak:latest
docker pull mattermost/mattermost-team-edition:latest
docker pull postgres:16-alpine
```

برای Jitsi، فایل release رسمی را دانلود کن و یک کپی نگه دار.

## DNS

- دامنه تمدید شده باشد
- nameserver ها تنظیم شده باشند
- A record ها ساخته شده باشند
- proxy mode تست شده باشد
- حالت direct DNS برای سرویس‌های UDP تست شده باشد

## کاربران

- کاربران Mattermost ساخته شده باشند
- کلید ادمین TeamSpeak ذخیره شده باشد
- پسورد ادمین FileBrowser تغییر کرده باشد
- stream key در Owncast تغییر کرده باشد
- لینک Jitsi حداقل با دو نفر تست شده باشد
