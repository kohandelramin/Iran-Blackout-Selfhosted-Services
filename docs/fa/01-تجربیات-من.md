# تجربه سرور من

این ریپوزیتوری را مثل یادداشت‌های شخصی خودم نوشتم. هدفش هم صرفا به اشتراک گذاشتن تجربیات خودم به صورت یک پکیج کامل طی این 2 ماه قطعی کامل بود که در هر صورتی هر کدوم از این بخش‌ها به دردتون خورد استفاده کنید.

اینها همه حداقلی‌ترین و راحت‌ترین سرویس‌هایی بودن که تونستم طی این مدت تونستم با وجود قطعی کامل اینترنت راه اندازی کنم.
- جلسه آنلاین با Jitsi
- چت صوتی با TeamSpeak 3
- چت روم با Mattermost
- استریم ساده با Owncast
- فایل شیرینگ با FileBrowser

قسمت سخت همیشه خود سرویس‌ها نبود. بیشتر دردسرها اطرافشان بود: Docker image دانلود نمی‌شد، DNS درست جواب نمی‌داد، SSL و CDN redirect مشکل می‌ساخت، یا کاربران از ISPهای مختلف تجربه متفاوتی داشتند.

## چیزی که برای من بهتر جواب داد

- همه چیز Docker based باشد.
- هر سرویس پوشه خودش را داشته باشد.
- Nginx reverse proxy تا جای ممکن روی host باشد.
- UFW rule ها واضح باشند.
- دامنه `.ir` و ساب‌دامین‌ها از قبل آماده باشند.
- تنظیمات mirror ها محلی ذخیره شده باشند.
